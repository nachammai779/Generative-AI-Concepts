## 🌟 Overview

Generative AI is transforming every industry, but how do these models actually work? This repository is a curated collection of interactive Jupyter notebooks that implement the foundational architectures behind generative models from scratch.

Each notebook is designed to be self-contained, richly commented, and focused on building intuition through code. No pre-trained models, no hidden magic — just the core algorithms laid bare for deep understanding.

---

## 📂 What's Inside

### 🔍 Attention Mechanism from Scratch
> `Attention_Mechanism.ipynb`
>
> The mechanism that revolutionized NLP and gave birth to the Transformer architecture. This notebook implements the **attention mechanism** step by step — computing query, key, and value vectors, calculating attention scores, and applying weighted context aggregation. Understand why attention is truly all you need.
>
> ### 📝 Beginner's Guide to Text Generation using LSTMs
> > `Beginners_Guide_to_Text_Generation_using_LSTMs.ipynb`
> >
> > A gentle introduction to **text generation** using Long Short-Term Memory networks. This notebook walks through the complete pipeline — from character-level tokenization to training an LSTM that learns to generate text character by character. Perfect for understanding how recurrent models capture sequential patterns in language.
> >
> > ### 🎭 GAN Face Generation
> > > `GAN_faces.ipynb`
> > >
> > > Step into the world of **Generative Adversarial Networks** — the framework where a generator and discriminator compete to produce increasingly realistic outputs. This notebook implements a GAN architecture designed to generate human faces, demonstrating the adversarial training process that has revolutionized image synthesis.
> > >
> > > ### 🌐 RNN-LSTM Neural Machine Translator
> > > > `GenAI_RNN_LSTM_Translator.ipynb`
> > > >
> > > > Build a **sequence-to-sequence neural machine translation** system using RNNs and LSTMs. This notebook constructs an encoder-decoder architecture that learns to translate between languages, showcasing the power of recurrent networks for mapping variable-length input sequences to variable-length output sequences.
